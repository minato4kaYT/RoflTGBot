# Папка для изображений

## permissions.png

Поместите сюда изображение с инструкцией по выдаче разрешений боту.

**Путь:** `img/permissions.png`

**Размер:** рекомендуется до 2MB

**Формат:** PNG, JPG, JPEG

## Использование

Когда пользователь подключает бота без необходимых разрешений, бот автоматически отправит это изображение вместе с текстом уведомления.

Если изображение отсутствует, будет отправлен только текст.
